# PRIME STACK 게임 방법

PRIME STACK은 숫자가 적힌 테트로미노를 쌓으며 소수와 합성수를 구분하는 3분 점수 경쟁 게임입니다.

## 목표

- 가로 10칸을 채워 줄을 지웁니다.
- 줄 안의 소수 블록은 점수를 올리고 합성수 블록은 점수를 내립니다.
- 제한 시간 180초 동안 최대한 높은 점수를 만듭니다.
- 블록이 천장에 닿으면 즉시 게임이 끝납니다.

## 기본 조작

| 동작 | 키 |
|---|---|
| 좌우 이동 | `←` `→` |
| 소프트 드롭 | `↓` |
| 하드 드롭 | `Space` |
| 시계 방향 90° 회전 | `↑` 또는 `W` |
| 반시계 방향 90° 회전 | `Z` |
| 180° 회전 | `A` |
| 홀드 | `C` 또는 `Shift` |
| 현재 블록 폐기 | `X` |
| 감도·설정 | `F1` |
| 아이템 사용 | `1` `2` `3` 또는 화면 슬롯 클릭 |

한글 입력 상태에서도 주요 이동 키 일부를 영문 키처럼 처리합니다.

## 점수 계산

### 블록 점수

- 줄에서 지워진 소수 블록: `+10점`
- 줄에서 지워진 합성수 블록: `−20점`
- 합성수 블록을 `X`로 폐기: `+30점`
- 소수 블록을 `X`로 폐기: `−30점`

### 줄·스핀·콤보

```text
줄 삭제 점수 = 지워진 블록 점수 합계 × 지운 줄 수 × 스핀 배율 + 콤보 보너스
```

- 동시에 2줄을 지우면 ×2, 3줄은 ×3, 4줄은 ×4가 적용됩니다.
- T-Spin과 잠김 상태 스핀은 ×2가 추가됩니다.
- 연속 줄 삭제의 두 번째 콤보부터 `+100`, `+200`, `+300`처럼 100점씩 증가합니다.
- 줄을 지우지 못하고 블록을 고정하거나 폐기하면 콤보가 끊깁니다.

여러 줄 배율은 소수의 양수 점수뿐 아니라 합성수의 음수 점수에도 적용됩니다.

## 난이도

| 난이도 | 숫자 범위 | 최종 점수 배율 |
|---|---:|---:|
| 쉬움 | 0~50 | ×0.5 |
| 보통 | 0~100 | ×0.75 |
| 어려움 | 0~200 | ×1.0 |
| 전문가 | 100~300 | ×1.3 |
| 마스터 | 200~600 | ×1.8 |

최종 점수는 `원점수 × 난이도 배율`을 반올림해 계산합니다.

## 입장

1. 학번과 이름을 입력합니다.
2. 싱글 또는 멀티 플레이를 선택하면 바로 게임을 시작할 수 있습니다.
3. 한 판이 끝난 뒤 다시 플레이하려면 처음 화면으로 돌아가 다시 입장하면 됩니다.

## 멀티플레이

1. 공개 방을 만들거나 방 목록에서 원하는 방에 참가합니다.
2. 대기 화면에서 방장이 일반전 또는 아이템전을 선택합니다.
3. 매칭이 완료되면 각자 난이도에 투표합니다.
4. 같은 난이도를 선택하면 그대로 확정되고, 다르면 두 난이도 중 하나를 돌림판으로 결정합니다.
5. 두 플레이어가 준비를 완료하면 카운트다운 후 시작합니다.

시간 종료 시 점수가 높은 플레이어가 승리하며, 먼저 천장에 닿은 플레이어는 패배합니다.

## 아이템전

지운 모든 줄의 모든 블록이 소수일 때 무작위 아이템 1개를 얻습니다. 최대 3개를 보관하며 `1` `2` `3` 키 또는 슬롯 클릭으로 사용합니다.

| 아이템 | 효과 |
|---|---|
| 먹물 | 상대 화면을 10초 동안 넓게 가립니다. |
| 초고속 | 상대 블록의 자동 낙하 속도를 10초 동안 3.2배로 만듭니다. |
| 좌우 반전 | 상대의 좌우 이동키를 10초 동안 반대로 만듭니다. |
| 홀드 봉인 | 상대가 10초 동안 홀드를 쓰지 못하게 합니다. |
| NEXT 차단 | 상대의 다음 블록 표시를 10초 동안 숨깁니다. |
| 무게추 | 상대가 10초 동안 하드 드롭 외 조작을 하지 못하게 합니다. |
| 방어막 | 다음 방해 아이템을 한 번 막습니다. |
| 정화 | 현재 적용된 모든 방해 효과를 즉시 제거합니다. |

## 플레이 팁

- 소수만으로 줄을 완성하면 안정적으로 점수를 얻고 아이템전에서는 아이템도 받습니다.
- 합성수 블록이 줄에 들어가기 전에 `X`로 폐기하면 점수를 얻을 수 있습니다.
- 높은 난이도는 최종 배율이 높지만 소수 판별 범위가 넓어집니다.
- 멀티에서는 방어막·정화를 위험한 순간까지 아껴두는 것도 좋습니다.
